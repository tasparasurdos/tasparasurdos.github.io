Introdução

A API do Catálogo de Tecnologias Assistivas para Surdos fornece acesso programático aos dados do catálogo, permitindo que desenvolvedores integrem as informações em suas próprias aplicações. Todos os endpoints são acessíveis via HTTP GET e retornam dados no formato JSON. O objetivo é facilitar o acesso a informações sobre tecnologias assistivas voltadas para surdos, organizadas em diferentes formatos e agrupamentos.

Como Usar

Como esta é uma API estática, você pode fazer requisições diretamente usando ferramentas como fetch em JavaScript ou requests em Python. Abaixo estão exemplos de como acessar os dados:

JavaScript (fetch)

fetch('https://tasparasurdos.github.io/api/leitor-tela.json')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Erro:', error));
Python (requests)

import requests
response = requests.get('https://tasparasurdos.github.io/api/leitor-tela.json')
data = response.json()
print(data)

Substitua leitor-tela.json pelo slug da tecnologia desejada, que pode ser obtido no endpoint /indice.json.

Estrutura de Dados

Cada tecnologia individual contém os seguintes campos:

  • titulo: String - Nome da tecnologia
  • descricao: String - Descrição detalhada da tecnologia
  • orientacao: String - Instruções de uso
  • dicas: String - Dicas para melhor uso
  • etapas_justificativa: String - Justificativa pedagógica
  • imagem: String - URL da imagem (ex.: https://tasparasurdos.github.io/imagens/tec1.jpg)
  • categoria: String - Categoria da tecnologia
  • custo: String - Custo (ex.: "Gratuito", "Pago")
  • requer_internet: String - "Sim" ou "Não"
  • plataformas: String - Plataformas suportadas (separadas por vírgula)
  • autor: String - Autor da tecnologia
  • link: String - URL para mais informações
  • etapas: String - Etapas de ensino (separadas por ponto e vírgula)

GET /<slug>.json
Descrição

Retorna os detalhes completos de uma tecnologia assistiva específica, identificada pelo seu slug. O slug pode ser obtido consultando o endpoint /indice.json.

URL

https://tasparasurdos.github.io/api/.json

Parâmetros

Nenhum

Exemplo

Para a tecnologia com slug "tec1":

https://tasparasurdos.github.io/api/tec1.json

  "descricao": "Descrição detalhada da tecnologia.",

  "etapas_justificativa": "Justificativa das etapas.",
  "imagem": "https://tasparasurdos.github.io/imagens/tec1.jpg",
  "categoria": "Categoria 1",
  "custo": "Gratuito",
  "requer_internet": "Sim",
  "plataformas": "Web",
  "autor": "Autor da tecnologia",
  "link": "Link para mais informações",
  "etapas": "Ensino Fundamental"
}
